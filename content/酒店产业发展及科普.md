# 酒店产业发展及科普观察手册

## 1. 行业结构

- **需求侧**：商务出差、旅游出行、本地短住、考试培训、临时休息等住宿需求。
- **供给侧**：单体酒店、经济型连锁酒店、中端酒店、中高端酒店、豪华酒店、民宿和长租公寓。
- **渠道侧**：OTA 平台、酒店集团自有会员体系、企业差旅协议、线下自然客流。
- **运营侧**：物业、装修、客房服务、布草洗涤、前台系统、收益管理、供应链采购。

## 2. 商业模式

- **直营 / 自有模式**：集团自己投入物业、装修和运营，控制力强，但扩张慢、资本占用高。
- **租赁经营模式**：租用物业后自行装修运营，资产较轻，但仍承担租金、装修和经营风险。
- **管理加盟 / 特许经营模式**：加盟商出资和经营，集团输出品牌、标准、系统和供应链，适合快速扩张。

## 3. 关键公司

- **华住**：汉庭、全季、橘子、海友等品牌，特点是标准化、会员体系和 IT 系统能力强。
- **锦江**：国内酒店集团巨头，拥有较大门店规模。
- **首旅如家**：国内经济型和中端连锁酒店重要玩家。
- **万豪 / 希尔顿**：国际酒店集团，对比对象通常是全球品牌、会员体系和高端酒店管理能力。

## 4. 核心能力

- **标准化**：让不同城市、不同门店提供相对稳定的住宿体验，降低用户决策成本。
- **模块化**：把装修和设备拆成可复制组件，提高开店速度，降低施工和维护成本。
- **供应链压价**：依靠大规模门店需求集中采购，压低床垫、布草、卫浴和设备成本。
- **会员直连**：减少 OTA 佣金，积累用户数据，提高复购和价格控制能力。
- **动态定价**：根据供需、节假日、库存、竞品和渠道数据调整价格，提高单房收益。
