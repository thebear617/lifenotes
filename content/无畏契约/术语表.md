# 术语表

> 本文件用于无畏契约视频转录后的术语校正，不是游戏百科。使用时先保留原始 `.srt` / `.txt` / 初始 `.md`，再在整理阅读版笔记或归档到 [[无畏契约]] 前进行人工或 Agent 辅助校正。

## 使用规则

1. 先读原始转写上下文，再按本表校正；不要只按单个错词机械替换。
2. 高置信词可以直接统一到“标准写法”，低置信词先放进 `## 暂不确定词表`。
3. 同一个错词如果可能对应多个术语，必须依靠角色、技能、点位、对局动作判断。
4. 整理战术笔记时优先保证语义正确，其次再追求口语表达顺滑。
5. 新视频中反复出现的新错词，确认后追加到对应分类表。
